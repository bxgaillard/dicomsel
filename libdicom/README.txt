/libdicom

This directory contains the DICOM reader library, used within DicomSel.  It
has been written by Johan Moreau and is to be released under the GNU Library
General Public Licence (LGPL) very soon.  Its licence isn't defined for the
time being.

Some modifications and improvements were done during the development of
DicomSel.  They are listed in the "CHANGES.txt" file.
